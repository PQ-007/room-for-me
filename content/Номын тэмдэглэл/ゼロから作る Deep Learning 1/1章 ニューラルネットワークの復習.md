### 1.1.1 Вектор ба Матриц

Вектор бол хэмжээ болон чиглэлтэй байдаг хэмжигдэхүүн. Векторыг нэг хэмжээст цуваагаар илэрхийлж болох бол Матрицыг хоёр хэмжээст цуваагаар илэрхийлж болно. 

> Вектор, матрицыг  өргөтгөн N хэмжээст тооны олонлог ч үүсэж болно. Үүнийг ерөнхийд нь **ТЕНСОР** гэдэг.

```python
>>> import numpy as np
>>> x = np.array([1, 2, 3])
>>> x.__class__ # クラス名を表示
<class 'numpy.ndarray'>
>>> x.shape
(3,)
>>> x.ndim
1
>>> W = np.array([[1, 2, 3], [4, 5, 6]])
>>> W.shape
(2, 3)
>>> W.ndim
2
```

### 1.1.2 Матрицын элемент дээрх үйлдлүүд

>要素ごとの」とは英語で element-wise と言います。



```python
>>> W = np.array([[1, 2, 3], [4, 5, 6]])
>>> X = np.array([[0, 1, 2], [3, 4, 5]])
>>> W + X
array([[ 1, 3, 5],
[ 7, 9, 11]])
>>> W * X
array([[ 0, 2, 6],
[12, 20, 30]])
```

**Нэмж хасах үржиж хуваах үйлдлүүдийг хийхдээ зөвхөн харгалзах элемент / утгууд дээр шууд хийдэг.**

### 1.1.3  Броадкастинг

Хоорондоо ижил хэмжээст биш матрицууд дээр ч гэсэн аритметик үйлдлүүд хийж чадна. 2 х 2 хэмжээст матрицыг 10 гэх скаляр утгаар үржүүлснийг доор үзүүлэв. Ерөнхий тогтмол тоогоор гишүүн бүрийг үржүүлж байгаа хэрэг.

```python
>>> A = np.array([[1, 2], [3, 4]])
>>> A * 10
array([[10, 20],
[30, 40]])
```

Энэ мэтээр математик дээр бидний хийдэг матриц дээрх үйдлүүдийг numpy сан нөхөж өргөтгөн харгалзан үйлдлийг гүйцэтгэх байдлаар хийдэг. 

*Компьютер шууд хүн шиг харгалзуулаад үржүүлэхэд давталт ашиглах нэмэлт хэрэгцээ гарах учир аль хэдийн ижил хэмжээст матрицууд дээрх үйлдэл хийх методыг ашиглах үүднээс ингэж өргөтгөдөг юм шиг байна. *

Ингэж нэг матрицыг нөгөөд тааруулж өргөтгөж байгааг **броадкастинг** гэдэг.

 **Дүрэм:** 
1. ***Ижил мөр баганын тоотой бол асуудалгүй үйлдэл хийж болно.***
2. ***Эсвэл мөр багана хоёрын нэг нь нэгтэй тэнцүү байх үед ( 1 x m эсвэл n x 1 матрицыг өргөтгөөд том матрицын n x m хэмжээст хүргэж болох учир) Броадкастинг хийж харгалзуулан үйдлүүдийг хийнэ.***
3. ***Өөр тохиолдолд үйлдэл хийх боломжгүй.***

### 1.1.4 Вектор үржвэр ба Матриц үржвэр

$$\underbrace{x \cdot y}_{\text{Dot Product}} = \sum_{i=1}^{n} x_i y_i \tag{1}$$
>Векторын үржвэр нь “хоёр вектор нэг чиглэлд хэр зэрэг чиглэж харж байгааг” тоогоор илэрхийлсэн хэмжүүр юм. Яг нэг чиглэл рүү харж байвал вектор үржвэр 1, эсрэг харж байвал -1, хоорондоо 90 градус өнцөг үүсгэсэн / перпендикуляр байвал 0 байдаг.

```python
# ベクトルの内積
>>> a = np.array([1, 2, 3])
>>> b = np.array([4, 5, 6])
>>> np.dot(a, b)
32
# 行列の積
>>> A = np.array([[1, 2], [3, 4]])
>>> B = np.array([[5, 6], [7, 8]])
>>> np.dot(A, B)
array([[19, 22],
[43, 50]])
```

 Вектор үржвэр ч бай Матриц үржвэр ч бай `numpy.dot()` гэдэг функц ашиглан олж болно.

```python
A * B        # element-wise (энгийн үржих зөвхөн харгалзах элементүүдэд)
A @ B        # matrix multiplication (өөр нэг бичиглэл)
A.dot(B)    # matrix multiplication 
```

$$

\;\longrightarrow\;
