## **１課**　～時・～直後に


| 文法                     | 説明                            | 接続                     | 例文                        |
| ---------------------- | ----------------------------- | ---------------------- | ------------------------- |
| **～際に**                | ～とき（*硬い表現*）                   | *名詞*【の】・*動詞*【辞書形】＋*際に* | この整理券は、商品受け取りの**際**、必要です。 |
| **～に際して・～にあったて**       | ～するとき（*硬い表現*）                 |                        |                           |
| **～たとたん（に）**           | ～したら、直後に*意外なこと*が起こる           |                        |                           |
| **～（か）と思うと・～（か）と思ったら** | ～の後、すぐに続いて次の*出来事*や*大きな変化*が起こる |                        |                           |
| **～か～ないかのうちに**         | ～が終わると*同時に*、次のこと起こる           |                        |                           |
|                        |                               |                        |                           |
